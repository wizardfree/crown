# crown 👑

crown is a watermark generator written in Python

## Installation

Grab the latest version of the script from this repo.

```
$ git clone https://
$ cd crown
$ sudo chmod +x crown.py
$ 👍
```

## Usage

```
$ ./crown.py -i imagetobewatermarked.png -o ~/Downloads
```
